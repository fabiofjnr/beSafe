**beSafe** é um aplicativo desenvolvido para oferecer suporte emocional e um espaço seguro para pessoas que enfrentam bloqueios emocionais, pessoas com deficiência, vítimas de violência, e todos que buscam apoio para expressar seus sentimentos.

### 📲 Funcionalidades

- **Converse com nossa IA**: um chat confidencial com uma inteligência artificial que simula uma psicóloga online, ajudando você a desabafar e encontrar apoio.
- **Rede de Apoio**: compartilhe suas experiências e receba apoio de uma comunidade empática.
- **Segurança**: seu bem-estar e privacidade são nossa prioridade. A plataforma oferece um ambiente seguro para todas as interações.

### 🌈 Visual

O beSafe utiliza uma paleta acolhedora e relaxante, com destaque para a cor **#EBADFA**, e a tipografia **BreeSerif**, promovendo uma experiência visual confortável e amigável.
